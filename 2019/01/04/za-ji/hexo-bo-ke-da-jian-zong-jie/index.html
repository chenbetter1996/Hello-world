使用Hexo搭建博客总结
hexo
杂记
发布日期:   2019-01-04
文章字数:   232
阅读时长:   1 分

重要操作

  1. 将source下的个人md博客文件生成html、css、js到public目录下
    # generate 可缩写为g，打开localhost:4000
hexo generate

  1. 部署到本地服务器，可做测试

    # server 可缩写为s
hexo server

  2. 关联到github.io仓库

    1. 安装deploy git
      # 这个是：hexo 和git自动对接上传的关键组件
npm install hexo-deployer-git --save
    2. 修改_config.yml全局配置文件的部署仓库地址
      deploy:
type: git
repository: https://github.com/chgl16/chgl16.github.io.git
branch: maste

  3. 将本地博客(主要是public文件夹的内容)部署到github.io

    # deploy 可缩写为d, 需要清空缓存，不然提交的不是更新的
hexo clean  
hexo deploy -g

  4. hexo博客站点美化
