TensorFlow分类

TensorFlow分为纯CPU和使用GPU（比如有NVIDIA显卡）两种，类库会不一样。

电脑没有英伟达显卡，这里安装仅支持CPU的TensorFlow。

TensorFlow安装

安装要先保证已经安装了python环境，pip

  1. 使用pip原生安装

    # 纯CPU版
pip3 install --upgrade tensorflow

    # GPU版
pip3 install --upgrade tensorflow-gpu

    会自动下载其他依赖

  2. 使用Anaconda进行安装

    这个要去其官网下载这个工具，然后使用它安装TensorFlow
