快捷键

内容操作快捷键

  • 复制当前行 Crtl + D
  • 删除当前行 Crtl + X 或者 Crtl + Y
  • 本文件内容搜索 Crtl + F
  • 本文件内容搜索替换 Crtl + R

  • 项目全局文件内容搜索 Crtl + Shift + F

  • 内容右对齐 Tab

  • 内容左对齐 Shift + Tab

窗口快捷键

  • 打开/隐藏左侧项目目录结果 Alt + 1 (小键盘数字无效)
  • 运行窗口 Alt + 4
  • 命令行终端 Alt + F12
